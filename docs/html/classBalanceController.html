Detailed Description

Balance Controller.

Like in life, so to in games do we need balance In order to maintain consistency in a game, you need a universal scale in which you operate from. This module creates a mathematical scalar curve that all statistics can be derived from.

Note
\(y = δ\)^[ \(x/(π^π)\)]
\(δ :=\) Constant based on game difficulty
\(x :=\) Character's Current Level
\(π :=\) Pi....mmmm Pi
\(y :=\) The scalar value
