\doxysubsection{Detailed Description}
\doxylink{class_config_manager}{Config\+Manager} manages configuration loading and updating throughout the engine. 

In order to handle the number of customizable options for the engine, a config file system has been implemented. This class will manage the configuration file and return it\textquotesingle{}s content to the engine. \begin{DoxyNote}{Note}
values can be reloaded without having to restart the engine. 
\end{DoxyNote}
